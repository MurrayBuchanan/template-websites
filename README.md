# Template Websites

This repo contains a collection of website designs I made when I was younger, with the original content removed. Feel free to use them if you find them useful.

The code is left largely as it was originally written and has not been reviewed since, aside from removing personal details. With how far AI has come, you might prefer to build something from scratch now, but these may still be useful as a starting point or for inspiration.

### [Typewriter](typewriter-portfolio/)
A simple typewriter-style personal site template, originally built as one of my first projects while learning how to publish websites.

Created July 2021

### [Bouncing Portfolio](bouncing-portfolio/)
Made as more of a portfolio intro or launch screen. A bit more arty than the others, so it suits a minimal personal site with only a small amount of information.

Created August 2021

### [Card Style Portfolio](card-style-portfolio/)
A visual portfolio template with a card-style layout, designed for a short personal intro, contact links, and a small selection of featured projects.

Created December 2021

### [Parent Company](parent-company/)
A minimal template inspired by Alphabet, designed as a landing page for a holding company.

Created August 2021

### [CV Template](cv-template/)
A minimal CV template built for showing experience, roles, and career history in a straightforward single-page layout.

Created April 2022

### [Forwarding Page](forwarding-page/)
A simple forwarding page for directing visitors to a main website or current project page. Originally made to avoid duplicating sites across university and work projects while keeping links up to date.

Created December 2023 - Still in use
