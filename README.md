# Template Websites

This repo contains a collection of website designs I made when I was younger, with the original content removed. Feel free to use them if you find them useful.

With how far AI has come, you might prefer to make your own from scratch now, but these may still be useful as a starting point or inspiration.

### [Typewriter](typewriter-portfolio/)
Created as my first website inspired by Microsoft
Created July 2021


### [Bouncing Portfolio](bouncing-portfolio/)
Designed as a launch screen for a portfolio, more arty than others for when you have little information you want to put in there


Created August 2021


### [Forwarding Page](forwarding-page/)
Created as a forwarding address of all the of websites notably as the root directory for any University and Work projects associated with me to avoid having to created duplicate sites, while allowing people to get up-to-date info.
